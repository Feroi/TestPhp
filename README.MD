# **Requirements**
- PHP >= 7.3
- Symfony >5.*

# **Setup**
1 - Install all dependencies :

~~~
    composer install
~~~

Run:
~~~
    symfony server:start
~~~
or
~~~
    cd public
    php -S 127.0.0.1:8000
~~~
